Política de privacidade

Última atualização: maio de 2026

Quais dados o MathQuest coleta

O MathQuest é um jogo educacional que não coleta dados pessoais identificáveis dos alunos. Os dados guardados são:

  • Apelido — o nome que o aluno escolhe usar no jogo. Pode ser uma forma de tratamento ("Lu", "João do 7ºA") ou inventado. Não pedimos o nome completo.
  • Identificador anônimo — um código aleatório (UUID) gerado pelo Supabase Auth, sem ligar a qualquer dado pessoal do aluno.
  • Progresso de jogo — quantas fases foram concluídas, quantas estrelas, XP, conquistas e a data de cada atualização.
  • Código de turma (opcional) — se o aluno digitou um código dado pelo professor, o vínculo aluno↔turma é salvo para o professor poder acompanhar o progresso.

Para professores que criam conta no painel:

  • E-mail e senha (a senha é armazenada criptografada pelo Supabase, não temos acesso à senha em texto).
  • Lista de turmas criadas e o código de cada uma.

Quem vê o quê

O banco de dados usa Row-Level Security (RLS) do Supabase Postgres, o que significa que regras são impostas no servidor:

  • Aluno só vê (e escreve) o próprio progresso. O navegador devolve 403 se tentar acessar o progresso de outro.
  • Professor só vê as turmas que ele mesmo criou e o progresso dos alunos vinculados àquelas turmas.
  • Mais ninguém tem acesso. Não vendemos, não compartilhamos, não anunciamos.

Cookies, trackers, anúncios

O MathQuest não usa cookies de rastreamento, Google Analytics, pixels do Facebook nem qualquer rede de anúncios. Os dados em localStorage ficam no próprio aparelho do aluno e servem apenas para fazer o jogo funcionar offline.

LGPD e dados de menores

Reconhecemos que parte significativa dos usuários são crianças e adolescentes. Por isso:

  • Não pedimos nome completo, e-mail, telefone, foto ou qualquer dado que possa identificar pessoalmente o aluno.
  • O professor é responsável por obter consentimento dos responsáveis dos alunos antes de pedir que joguem o MathQuest na escola.
  • Apelidos são pseudonimizados — recomendamos não usar nome completo nem dados sensíveis no campo "apelido".

Direitos

Você (ou o responsável pelo aluno menor) pode pedir a exclusão do progresso e da conta a qualquer momento. Como não temos formulário de contato ainda, faça a solicitação via issues do repositório no GitHub.

Onde os dados ficam

Os dados rodam no Supabase (serviço gerenciado de Postgres). A instância está hospedada na região South America (São Paulo) — então seus dados ficam no Brasil.

Mudanças nesta política

Se mudarmos algo importante aqui, atualizamos a data no topo. O histórico fica visível no histórico de commits deste arquivo.
